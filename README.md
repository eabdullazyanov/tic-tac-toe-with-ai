## Basic Tic Tac Toe game with AI

- Supports modern browsers including mobile browsers;
- Implemented the Minimax algorithm for AI.

### Used technologies
- React w/ hooks
- ESLint
- [shortid](http://localhost:3000)
- GitHub Pages for deployment
